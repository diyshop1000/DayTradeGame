📈 デイトレード練習ゲーム

架空の1,000万円を元手に、株式トレードを疑似体験できるゲームです。

【遊び方】
実在銘柄の過去の日足データ（CSV）を読み込んでゲームを開始します。
CSVの取得はセキュリティ上サーバー側では行えないため、
各自のローカル環境でダウンロードしてからアップロードしてください。

【データ取得について】
・1日3回まで取得可能
・デフォルトは指定銘柄の直近1年分（約245営業日）
・期間指定も可能なので、過去の難しい相場を選んで何度でも練習できます
・読み込みの安定性から1〜2年分がおすすめです

【ゲームの流れ】
1年分のチャートを最短250秒で体験。
10秒/本〜1秒/本の速度で調整できます。

【売買操作】
現物買い・現物売り・信用売り・全ポジ決済のシンプル設計。

【デイトレゲームが全て無料な理由】
有料配布にすると事業とみなされて金融庁に事業者登録しないと
金融商品取引法違反になるからです。
